# 広告クリエイティブ生成プロンプト

このドキュメントは、広告オークションサービスの広告クリエイティブを作成するための画像生成AIプロンプトです。

## プロンプト バージョン1: モダン・デジタルスタイル

### 基本プロンプト（英語）
```
Create a professional digital advertisement banner for a blockchain-based advertising auction platform. The image should convey the message "Your ads can now be displayed" or "Advertise here now". 

STYLE & AESTHETICS:
- Modern, clean, minimalist design with a tech-forward aesthetic
- Gradient background: deep blue (#1a1a2e) transitioning to vibrant purple (#16213e) with subtle teal accents (#00d9ff)
- Glassmorphism effects: translucent panels with frosted glass appearance
- Soft neon glow effects on key elements
- Professional, sleek, and futuristic appearance

COMPOSITION:
- Horizontal banner format, 16:9 aspect ratio
- Central focal point: a stylized megaphone or billboard icon with digital/blockchain elements
- Floating particles or data streams in the background suggesting blockchain technology
- Diagonal composition with dynamic flow from top-left to bottom-right
- Golden ratio positioning for key visual elements

KEY VISUAL ELEMENTS:
1. Main icon/illustration (center):
   - Stylized digital billboard or megaphone
   - Integrated with blockchain chain links or nodes
   - Subtle animation-ready appearance (static but suggesting movement)
   - Modern flat design with 3D depth through shadows
   - Size: occupies approximately 30-40% of the canvas

2. Text/messaging area:
   - Bold, sans-serif typography (futuristic font style)
   - Primary message: "広告を出せるようになりました" (Japanese) or "Your Ads, Your Way" (English)
   - Secondary message: "Advertise with Crypto" or "Blockchain-Powered Advertising"
   - Text with subtle glow effect matching the color scheme
   - Position: bottom-right or center-right

3. Blockchain/tech elements:
   - Subtle chain links connecting nodes (not too prominent)
   - Digital grid pattern in background (very subtle)
   - Floating geometric shapes suggesting data flow
   - Hexagonal patterns (subtle, behind main elements)
   - Sparkles or light particles around main icon

COLOR PALETTE:
- Primary: Deep blue (#1a1a2e, #0f3460)
- Secondary: Vibrant purple (#16213e, #533483)
- Accent: Electric teal/cyan (#00d9ff, #00f5ff)
- Highlight: Bright yellow/gold (#ffd700, #ffed4e) for call-to-action elements
- Text: White (#ffffff) with subtle glow
- Shadows: Deep navy (#000000, opacity 30%)

TECHNICAL SPECS:
- Resolution: 1920x1080px (or 3840x2160px for high-res)
- Format: PNG with transparent background option
- Quality: High detail, sharp edges, professional finish
- Depth: Multi-layered with clear foreground, middle ground, and background separation
- Lighting: Soft, directional lighting from top-left, creating depth
- Shadows: Subtle, soft drop shadows on main elements

MOOD & ATMOSPHERE:
- Positive, empowering, and forward-thinking
- Professional yet approachable
- Innovation and technology-forward
- Trustworthy and reliable
- Exciting and opportunity-filled

AVOID:
- Cluttered composition
- Overly complex blockchain visuals
- Dark or negative imagery
- Outdated design elements
- Low contrast or hard-to-read text
- Copyrighted logos or brands

ADDITIONAL DETAILS:
- Include subtle Japanese characters or elements if bilingual version needed
- Ensure text area has enough space for readability
- Make it suitable for both web and print (if needed)
- Consider mobile responsiveness (text should be readable when scaled down)
```

### 日本語プロンプト版
```
ブロックチェーン基盤の広告オークションプラットフォーム用のプロフェッショナルなデジタル広告バナーを作成してください。画像は「広告を出せるようになりました」または「今すぐ広告を出稿」というメッセージを伝える必要があります。

スタイルと美学：
- モダンでクリーン、ミニマルなデザイン、テクノロジー重視の美学
- グラデーション背景：深い青（#1a1a2e）から鮮やかな紫（#16213e）へ、控えめなティールアクセント（#00d9ff）
- グラスモーフィズム効果：半透明パネル、すりガラス風の外観
- 主要要素にソフトなネオングロー効果
- プロフェッショナルで洗練された未来的な外観

構図：
- 横型バナー形式、16:9のアスペクト比
- 中央焦点：デジタル/ブロックチェーン要素を持つスタイライズされたメガホンまたは看板アイコン
- 背景に浮遊する粒子やデータストリーム（ブロックチェーン技術を示唆）
- 左上から右下への動的な流れを持つ対角線構図
- 主要視覚要素の黄金比配置

主要視覚要素：
1. メインアイコン/イラスト（中央）：
   - スタイライズされたデジタル看板またはメガホン
   - ブロックチェーン鎖リンクまたはノードと統合
   - 控えめなアニメーション風の外観（静的だが動きを示唆）
   - 影による奥行きを持つモダンなフラットデザイン
   - サイズ：キャンバスの約30-40%を占める

2. テキスト/メッセージエリア：
   - 大胆なサンセリフタイポグラフィ（未来的なフォントスタイル）
   - 主要メッセージ：「広告を出せるようになりました」
   - サブメッセージ：「暗号通貨で広告を出稿」または「ブロックチェーン駆動の広告」
   - カラースキームに合わせた控えめなグロー効果を持つテキスト
   - 位置：右下または中央右

3. ブロックチェーン/技術要素：
   - ノードを接続する控えめな鎖リンク（目立ちすぎない）
   - 背景のデジタルグリッドパターン（非常に控えめ）
   - データフローを示唆する浮遊する幾何学的形状
   - 六角形パターン（控えめ、メイン要素の後ろ）
   - メインアイコン周辺のスパークルまたは光の粒子

カラーパレット：
- プライマリ：深い青（#1a1a2e, #0f3460）
- セカンダリ：鮮やかな紫（#16213e, #533483）
- アクセント：エレクトリックティール/シアン（#00d9ff, #00f5ff）
- ハイライト：明るい黄色/ゴールド（#ffd700, #ffed4e）CTA要素用
- テキスト：控えめなグローを持つ白（#ffffff）
- 影：深いネイビー（#000000、不透明度30%）

技術仕様：
- 解像度：1920x1080px（または高解像度の場合は3840x2160px）
- フォーマット：透明背景オプション付きPNG
- 品質：高詳細、シャープなエッジ、プロフェッショナルな仕上がり
- 深度：明確な前景、中景、背景の分離を持つ多層構造
- 照明：左上からのソフトな指向性照明、深度を作成
- 影：主要要素の控えめなソフトドロップシャドウ

ムードと雰囲気：
- ポジティブで力強く、前向き
- プロフェッショナルでありながら親しみやすい
- イノベーションとテクノロジー重視
- 信頼性が高く、信頼できる
- エキサイティングで機会に満ちている

避けるべき要素：
- 雑然とした構図
- 過度に複雑なブロックチェーン視覚要素
- 暗いまたは否定的なイメージ
- 時代遅れのデザイン要素
- 低コントラストまたは読みにくいテキスト
- 著作権のあるロゴやブランド

追加の詳細：
- バイリンガル版が必要な場合は、控えめな日本語文字や要素を含める
- テキストエリアに十分な読みやすさのスペースを確保
- Webと印刷の両方に適している（必要に応じて）
- モバイル対応を考慮（縮小してもテキストが読みやすく）
```

## 使用ガイドライン

### どのプロンプトを選ぶべきか

1. **バージョン1（モダン・デジタル）**: 
   - 一般的な広告主向け
   - バランスの取れたプロフェッショナルな外観
   - 推奨：初期の広告クリエイティブ

2. **バージョン2（イラスト・フレンドリー）**:
   - 小規模ビジネスや個人向け
   - 親しみやすく、わかりやすい
   - 推奨：幅広い広告主にアプローチ

3. **バージョン3（ミニマル・エレガント）**:
   - 高級ブランドやプレミアム広告主向け
   - 洗練された、上品な外観
   - 推奨：高額入札者向け

4. **バージョン4（テクノロジー・フューチャリスティック）**:
   - テック企業やイノベーター向け
   - 革新的で未来的な外観
   - 推奨：ブロックチェーン/Web3関連企業

### カスタマイズのヒント

- **テキスト**: プロンプト内の日本語テキストを任意のメッセージに変更可能
- **色**: カラーパレットをブランドカラーに合わせて調整
- **サイズ**: アスペクト比を変更（例：1:1、4:3、21:9など）
- **要素**: 特定の要素を強調または削除

### 生成AI別の推奨設定

**Midjourney**:
- `--ar 16:9` を追加
- `--style raw` または `--style 4` で詳細度を上げる
- `--v 6` で最新バージョンを使用

**DALL-E 3**:
- プロンプトをそのまま使用
- サイズ指定: "1920x1080 pixels"

**Stable Diffusion**:
- ネガティブプロンプトに以下を追加:
  ```
  low quality, blurry, distorted, ugly, bad anatomy, watermark, text errors, low resolution
  ```
- サンプリングステップ: 30-50
- CFG Scale: 7-9

## 注意事項

1. **著作権**: 生成された画像は商用利用可能か確認
2. **テキストの可読性**: 生成後にテキストが読みやすく、必要に応じて追加
3. **ブランド一貫性**: 複数のバナーを作成する場合は一貫性を保つ
4. **A/Bテスト**: 複数のバージョンを作成して効果を測定
