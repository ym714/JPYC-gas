# Funny Address Miner

面白いEthereumアドレスを並行処理で探索するGoアプリケーション

## 概要

このアプリケーションは、特定のパターンに一致するEthereumアドレスを効率的に探索するためのツールです。複数のゴルーチンで並行処理を行い、面白いアドレスを見つけます。

初期設定では、`0xE7C3`で始まるアドレスを探索します。

## 機能

- 並行処理による高速なアドレス探索
- カスタマイズ可能な面白いアドレス判定ロジック
- リアルタイムの進捗表示
- Ctrl+Cで安全に中断可能

## インストール

```bash
cd funnyaddress
go mod download
```

## 使い方

```bash
# デフォルト設定（4ワーカー）で実行
go run .

# ワーカー数を指定して実行
go run . -workers 8

# ビルドして実行
go build -o funnyaddress
./funnyaddress -workers 4
```

## カスタマイズ

### 面白いアドレス判定ロジックの変更

`checker.go`の`IsFunnyAddress`関数を編集することで、異なるパターンのアドレスを探索できます：

```go
func IsFunnyAddress(address string) bool {
    // 例: 末尾が特定の文字列で終わる
    return strings.HasSuffix(address, "0000")
    
    // 例: 特定の文字列を含む
    return strings.Contains(address, "CAFE")
    
    // 例: 特定の文字で始まる
    return address[:6] == "0xDEAD"
}
```
